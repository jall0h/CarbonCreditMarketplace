# CarbonTrade Frontend

This folder contains the Vercel-ready Next.js frontend for CarbonTrade.

## Run locally

```bash
cd frontend
npm install
npm run dev
```

## Deploy on Vercel

- Import this repository in Vercel.
- Set **Root Directory** to `frontend`.
- Keep framework as **Next.js**.

## Current scope

- Product narrative for CarbonTrade mission and architecture
- Sections for AI Carbon Score Engine, Blockchain Transparency Layer, and Marketplace
- Carbon Equity Index (CEI), problem framing, and technical component overview

## Planned backend integration

Backend remains unchanged for now, but the workspace is organized for upcoming work in `../backend`.
